# Personal Portfolio Website #

## Project Title ## 

Portfolio of ***Jahida Khanom Jinar***

## Project Description ##
A responsive personal portfolio website designed and developed to showcase my skills, projects, and achievements as a Full Stack Developer. This website meets all requirements from the lab task including a clean design, mobile responsiveness, and professional presentation of my work. It features interactive elements, smooth animations, and a downloadable CV in PDF format.

## Technologies Used ##

**1. HTML5** - Semantic markup and structure

**2. CSS3** - Custom properties, Flexbox, Grid, animations, responsive design

**3. JavaScript (ES6+)** - Interactive features and animations

**4. Font Awesome 6.4.0** - Icon library

**5. Google Fonts (Poppins)** - Typography

**7. Git & GitHub** - Version control and repository hosting

**8. Vercel** - Live deployment platform
